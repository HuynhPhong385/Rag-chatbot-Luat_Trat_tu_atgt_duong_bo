# Handwritten Digit Recognition

A desktop application for recognizing handwritten digits using **Python**, **Tkinter**, and a **CNN model trained on MNIST**.

---

## Features
- Draw digits directly on the canvas  
- Real-time prediction using a trained deep learning model  
- Clean Tkinter UI  
- Supports clearing and redrawing  
- Lightweight and easy to run locally  

---

## Model
The model is a Convolutional Neural Network (CNN) trained on the **MNIST handwritten digits dataset (28x28 grayscale images)**.

You can:
- Use the pre-trained model included in this repo  
- Or retrain the model using the provided training script  

---

## Installation

### 1. Clone the repository
git clone https://github.com/<your-username>/handwritten-recognition.git

### 2. Install dependencies
pip install -r requirements.txt
